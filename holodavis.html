Project Brief

A collaboration project with the Davis Museum at Wellesley College to create a HoloLens application that allows a user to curate their own exhibit using 3D scanned artifacts from the British Museum. Users are prompted with an artifact inventory, from which they can select objects. Once selected, they can adjust the size, rotation, and location of the object. They can also remove the object's material and show information about it. The aim of this project is to augment the classroom experience in object-based disciplines by bringing previously unavaible objects into the classroom to facilitate exploration and understanding of the object.

Problem Statement

Artifacts are restricted to certain spaces, usually museums, which makes it hard to study them in the classroom setting. There have been attempts to bring artifacts into the classroom setting through online websites such as SketchFab and using model replicas. However, there is still a lack of ability to actually bring the artifacts into the classroom.

This project aims to solve this by bringing artifacts into the classroom setting with Augmented Reality, by using the Microsoft HoloLens. We envisioned an application that allows a user to select a 3d model representation of the object from an inventory. Following this selection, the user can use air gestures and voice commands to manipulate the object. Through the application, the user can scale, rotate, and move the object in the space around them. They can also remove the outside paint layer of the object so that they can inspect the object's wear pattern, request to see more information, reset the size of the object, and delete it.

Figure 1. A user opening a virtual gallery to select an object to place in the space and a user customizing an exhibit by placing an artifact in the physical world (a museum)

This project brought forth new problems for User Experience Design since there is no research on how users interact in holographic spaces. Once an initial development, we tested the application with a pilot study of users. We gathered qualitative data from this study, translated it into design decisions, and amended the application to reflect the user feedback gathered. Namely, we changed the interface so that it reflected more information about the objects on the inventory and a side menu so that the user could directly manipulate the objects by navigating each object's menu.


Figure 2. Actual screenshots of the second iteration of the application


This second iteration was tested with a study of 60 participants, where the application was compared against 3D model replicas of the object and a desktop 3D modeling application (SketchFab).

We are currently extending the project to also augment the museum experience. A user will be able to use the HoloMuse application within the museum to have a "guided" tour and learn more about objects by viewing information through different mediums (audio, video, text).
