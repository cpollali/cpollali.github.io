Project Brief: A collaboration project with the Davis Museum at Wellesley College to create a HoloLens application that allows a user to curate their own exhibit using 3D scanned artifacts from the British Museum. Users are prompted with an artifact inventory, from which they can select objects. Once selected, they can adjust the size, rotation, and location of the object. They can also remove the object's material and show information about it. The aim of this project is to augment the classroom experience in object-based disciplines by bringing previously unavaible objects into the classroom to facilitate exploration and understanding of the object.

We are currently extending the project to also augment the museum experience. A user will be able to use the HoloMuse application within the museum to have a "guided" tour and learn more about objects by viewing information through different mediums (audio, video, text).
