TwitterTrails

Project Brief: TwitterTrails is a tool that allows members of the media to track the trustworthiness of stories shared on Twitter. The project has examined 457 stories so far. For each story, the algorithm measures how widely the story spread and how skeptical users are about its validity. By measuring crowd behavior, the TwitterTrails algorithm allows journalists to investigate claims and determine whether they appear to be true or false.
