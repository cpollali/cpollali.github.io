Ruhlman Data Science

Associated with: CS249 Final Project

Responsibilities:
  • Wrote a python script to parse the data from 2003 and 2004 and helped with overall data cleaning
  • Conducted analysis on the total number of presentation per unique student and determined who "won" the conference by having the most presentations
  • Collaborated with the other 19 students to produce overall conclusions based on all the data analysis done by the team
  • Presentor of the work at the Wellesley College Ruhlman Conference 2016
Time Frame: Spring 2016

Project Brief: This was the final projest for CS249, "Data, Analytics, and Visualization." It used all the twenty years of data from the Wellesley Ruhlman Conference to draw conclusions about the different departments at Wellesley and how the conference has changed over the years.

Figure 1. Top 23 words from the 20 years of Ruhlman Data

The first step in this project was to clean the last twenty years of data so that we could run tests on them. This data was in PDF files of scanned programs, which made it hard to run scripts on. Thus, they had to be converted into Word Documents and then we wrote a parser to go through the Word Documents and create spreadsheets. This was tricky because there were inconsistent formats throughout the years for the programs so each year's parser had to be written slightly differently so that we could end up with the same format in the final spreadsheet.

After having the clean data in spreadsheet form, we used Tidy Data so that we could start doing our analysis. This analysis included finding out the number of presentations each Wellesley College department has sponsored for the Ruhlman Conference over the years, the most popular majors at the conference over the years, whether a singular department, and whether a particular class year has dominated the conference, etc. Below are a few of the graphs that we came up with as a result of this data analysis.

By conducting this data analysis, we were able to draw conclusions about the Ruhlman Conference and the greater Wellesley College community as a whole. For example, there is no department that has consistently dominated the conference, there are more upperclass students who present at the conference, and the highest number of presentations per unique student have been 5. The code used behind this analysis has been aimed to be incorporated to the Ruhlman Conference website so people can continue to explore this data.
